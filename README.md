# ¿Quién finge?

Juego local del impostor por categorías, pensado para jugar desde un solo móvil.

## Desarrollo

```bash
pnpm install
pnpm dev
```

## Verificación

```bash
pnpm test
pnpm build
```

## Flujo actual

1. Configurar jugadores e impostores.
2. Elegir una o varias categorías.
3. Pasar el móvil para revelar cada rol.
4. Jugar la ronda y señalar al impostor en persona.
5. Empezar otra ronda sin repetir palabras recientes.
